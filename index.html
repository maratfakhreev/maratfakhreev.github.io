Open Source

  • Backbone-base
    Backbone.Marionette modern skeleton based on ES6/ES7, Backbone.Marionette, Browserify technologies.
  • React-base
    Kick-start your new web application based on ES6/ES7, React and Flux technologies.
  • Electron React Starter Kit
    Kick-start your desktop application for MacOS/Windows/Linux based on Electron, React and Flux technologies.
  • React Native Starter Kit
    Simple starter kit for mobile applications based on ES6, React Native and Flux technologies.
  • Backbone ES6 promise
    A plugin for Backbone.js that overrides default promises with modern ES6 promises.
  • Marionette animated region
    Backbone.Marionette animated region is convinient animated plugin based on Velocity.js. It provides you to create beautiful animation effects for your Marionette views.
  • Marionette routes helper
    Backbone.Marionette routes helper add helper methods for your Backbone.Marionette app routes.
  • Slush npm package generator
    A Slush generator for Node.js ES6 npm package. Includes ES2015 babel preset and babel polyfill, console and browser test environment (chai, mocha, phantomjs, eslint)
  • jQuery pixel perfect
    jQuery-Pixel-Perfect is a small plugin that conveniently allows you to transpose a design layout over a developed site. With this tool you do not need to make measurements and you can write CSS code that pixel-matches to the design layout.
